## Descrição e Motivo do projeto:
- Este projeto foi criado para simular uma roleta de cassinos reais. A roleta foi desenvolvida em JQuery e pode ser facilmente implementada em qualquer site/aplicação.
 
## Frameworks Usados:
- JQuery
